# Math Snake Adventure Documentation

## Project Overview
**Math Snake Adventure** is an educational browser game that combines the classic Snake gameplay with math drills. It features multiple themes, power-ups, and a tiered difficulty system suitable for children.

## Architecture
- **Type**: Pure Frontend (HTML/CSS/JS)
- **Hosting**: GitHub Pages / EdgeOne
- **State**: LocalStorage for persistence
